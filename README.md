# Food Delivery Robot with smart monitoring.

Project Overview
The Food Delivery Robot is designed to transport food safely within indoor environments such as hostels, colleges, offices, and hospitals. The robot is controlled using a NodeMCU microcontroller and operates through a Wi-Fi based interface.

# Objective
The objective of this project is to develop an automated robot that can deliver food efficiently while ensuring safety and real-time monitoring.

# Components Used

-NodeMCU Microcontroller
-Motor Driver Module
-DC Motors
-Servo Motor
-Smart Camera Module
-Power Supply Module
-Robot Chassis

# Working Principle

The robot operates using a NodeMCU microcontroller as the central control unit. When power is supplied, the NodeMCU connects to a Wi-Fi network and allows the user to control the robot remotely using an IP-based interface. The motor driver controls the DC motors to move the robot in different directions such as forward, backward, left, and right. A smart camera provides live video monitoring, and a servo motor secures the food compartment to prevent unauthorized access during delivery.

# Features

-Wi-Fi controlled robot
-Remote movement control
-Live video monitoring
-Secure food compartment
-Efficient indoor food delivery
